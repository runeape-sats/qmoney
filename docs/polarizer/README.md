# QMoney Polarizer Docs

This folder collects the docs related to the QMoney polarizer / physical-intuition demo.

## Files

- [`qmoney-pkey-polarizer-experiment.md`](qmoney-pkey-polarizer-experiment.md) — layman-friendly explanation of the current private-key QMoney path using a polarization thought experiment.
- [`polarizer-demo-evaluation.md`](polarizer-demo-evaluation.md) — evaluation memo covering what the demo does well, what it does not prove, and how to present it honestly.
- [`polarizer-demo-exhibit-script.md`](polarizer-demo-exhibit-script.md) — standard presentation script for live demos, booths, investor walk-throughs, and recorded explainers.
- [`polarizer-demo-script-variants.md`](polarizer-demo-script-variants.md) — audience-specific variants for founders/investors, researchers, and museum/public exhibit visitors.

## Positioning

These docs support the same core message:

> The polarizer demo is an intuition demo for QMoney's current **private-key** BB84-style path. It explains hidden basis choices, disturbance under wrong-basis probing, and why verification consumes the original note and requires remint. It is not yet a full hardware implementation or public-key quantum money.
