QMoney Oracle Logic Flow

How the current hidden-subspace toy verifier, TLA+ oracle model, and Z3 inductive checker line up across the flatter pkey_quorum/pubkey_hidden_subspace/tla/z3 layout.

Code-level verifier

  • • The Python prototype checks support over A and Hadamard support over A⊥.
  • • This is a toy public-key verifier, not an obfuscated oracle construction.
  • • The note family is intentionally separated from the BB84/quorum baseline.

TLA+ oracle model

  • • The spec abstracts away amplitudes and only tracks lifecycle + oracle consistency.
  • • Query actions append typed records to an oracle query log.
  • • Acceptance/rejection requires the oracle tables to exist for that serial.

Inter-state stability target

  • • TLC already checks finite-state invariant preservation under bounded queries.
  • • Z3 now checks symbolic one-step stability per transition.
  • • The checker lives at `z3/check_oracle_invariants.py`.
